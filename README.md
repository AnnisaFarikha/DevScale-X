# DevScale-X Bootcamp Progress

Repository ini berisi dokumentasi dan hasil pengerjaan selama mengikuti **DevScale-X Bootcamp**.  
Setiap folder merepresentasikan **progress harian (day-based)** sesuai materi dan tugas yang diberikan pada hari tersebut.

---

## Tujuan Repository

Repository ini dibuat untuk:
- Mencatat progres pembelajaran selama bootcamp
- Menyimpan hasil latihan, eksperimen, dan mini project
- Menjadi referensi perkembangan skill backend & fullstack

---

## Catatan Penting

- Repository ini bersifat **progress-based**, bukan final product
- Struktur dan isi folder bisa berkembang seiring bootcamp berjalan
- File seperti:
  - `node_modules`
  - database
  - `.env`
  
  **tidak di-upload ke repository**
